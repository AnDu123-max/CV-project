# 🎥 Real-Time Face Detection using OpenCV

## 📌 Overview
This project implements real-time face detection using Python and OpenCV. It uses a pre-trained Haar Cascade classifier to detect human faces through a webcam and draw bounding boxes around them.

---

## 🚀 Features
- Real-time face detection
- Uses webcam input
- Lightweight and fast
- No need for model training

---

## 🛠️ Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier
